# 🚀 NewGen Booking System

[Live Demo] 
## 📌 Overview

**NewGen Booking System** is a smart platform for managing appointments, hotel bookings and events. Features separate dashboards for **Customers**, **Providers** and **Admins**.

## ✨ Key Features

- **User Management** - Login/Registration, role-based access, profile pictures
- **Provider Management** - Business registration, verification, schedule management
- **Booking & Payments** - Real-time booking, payment tracking (Pending/Completed/Failed)
- **Admin Dashboard** - Statistics, charts, reports (CSV/PDF), audit log
- **Notifications** - Real-time updates for booking status changes
